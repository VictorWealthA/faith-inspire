# Faith Inspire

Faith Inspire is a Flutter application for browsing inspirational quotes,
affirmations, and scriptures with sharing, text-to-speech, favorites, and a
simple slideshow experience.

## Project Structure

The Flutter app now lives at the repository root.

Key directories:

- `lib/` application code
- `test/` widget tests
- `android/`, `ios/`, `macos/`, `linux/`, `windows/`, `web/` platform targets

## Development

Common commands:

```bash
flutter pub get
flutter analyze
flutter test
flutter run
```

Android debug build:

```bash
flutter build apk --debug
```
